HTML Reference Page

HTML

In order to specify a webpage to be in html you must first put <!DOCTYPE html> at the top of the code with a <html> tag below that, and at the end of all your code you must have </html> as an ending element

If you created a website with only these three tags with to writing in between it would be a blank page

Header

Some reason why to put a header into an html code is to give the page a title, add meta data, reference a style sheet, and more. But text that will show up in the header tag won't actually show up on the website.

The code for a header tag is <head> and you must also have a </head> at the end.

The code for the title tag is <title> and </title>. The title tag actually shows up inside of the header tags like this:

Body

The body element is where all of the text and pictures and links will show up on your webpage. Inside the body tag is where all of the content of your website will be.

The code for the body tags are <body> and </body> for the end tag. If you create a page with just the main html elements (<!DOCTYPE html>, html, header,) and you have an empty body element with nothing in between the tags, the website will still be blank

Comments

Comments are used as a way to take out parts of your code, so it doesn't display on the website, or to add comments to yourself so you can figure out what is going on if you come back to your code after a while.

The code for a comment element is <!-- for a starting tag, and --> for the end tag with whatever you want to say in the comment in between

The code for a link element is <a href ="URL of website"> and an end tag of </a>. You can type in text or even add apicture in between these tags so that when you click on the text it takes you to the specified website. Lists

There are multiple types of lists that you can create on a webpage. The two types of lists include unordered lists and ordered lists.

The code to make an unordered list is <ul> and </ul> and an ordered list is <ol> and </ol>. The lists elements are defined by the <li> and </li> tags. The list elements go inside the unordered or ordered list tags. This is an example of the two lists:

Tables

You can also create tables using html.

To define a table element, use the <table> and </table> tags. Inside each table is a set of rows. A row is defined by the <tr> and </tr> tags. Inside each row you can have two types of data. A table header, which defaults to bold and centered, and normal table data. A table header element is defined using the <th> and </th> tags while table data uses the <td> and </td> tags. You can also add styles to the table using CSS. Below is an example of a table.

Firstname Lastname Age
Jill Smith 50
Eve Jackson 94
John Doe 80

Blocks

A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can). However, an inline element does not start on a new line and only takes up as much width as necessary.
