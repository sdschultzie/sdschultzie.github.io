HTML

In order to specify a webpage to be in html you must first put <!DOCTYPE html> at the top of the code with a <html> tag below that, and at the end of all your code you must have </html> as an ending element. If you created a website with only these three tags with to writing in between it would be a blank page. Here is some example code:

<!DOCTYPE html>
<html>
<body>

</body>
</html>

Header

Some reasons why to put a header into an html code is to give the page a title, add meta data, reference a style sheet, and more. But text that will show up in the header tag won't actually show up on the website. The code for a header tag is <head> and you must also have a </head> at the end. Here is some example code.

<!DOCTYPE html>
<html>
<head>
  <title>My First HTML</title>
  <meta charset="UTF-8">
</head>
<body>

</body>
</html>

The title element does exactly what is sounds like. It defines a title for that webpage. However this title doesn't show up on the webpage itself but rather in the browser toolbar. The code for the title tag is <title> and </title>. Here is some example code:

<head>
  <title>This is the title of the document</title>
<head>

Body

The body element is where all of the text and pictures and links will show up on your webpage. Inside the body tag is where all of the content of your website will be. The code for the body tags are <body> and </body> for the end tag. If you create a page with just the main html elements (<!DOCTYPE html>, html, header,) and you have an empty body element with nothing in between the tags, the website will still be blank. Here is some example code.

<!DOCTYPE html>
<html>
<body>

</body>
</html>

Headings

The heading elements will actually show up on the web page. Headings are located inside the body element of your html code. Headings will be large and bold text, and there are 6 types of heading tags. The code for the heading tags are: <h1> and </h1> through <h6> and </h6>. The h1 element is considered 'more important' than the h6 element and will show up as larger and more bolded text than the h6 element. However both of these elements are 'more important' than the paragraph element. Here is some example code:

<!DOCTYPE html>
<html>
<body>
  <h1>This is a Heading</h1>
  <h2>This is a Heading</h2>
  <h3>This is a Heading</h3>
  <h4>This is a Heading</h4>
  <h5>This is a Heading</h5>
  <h6>This is a Heading</h6>
</body>
</html>

This example code would create a webpage that looks like this:

This is a Heading

This is a Heading

This is a Heading

This is a Heading

This is a Heading
This is a Heading

Paragraphs

the paragrpah element is located in the body tags and this text will also show up on the website, hust like most elements inside the body tags. A prargraph shows up as normal text and isn't bolded or enlarged. There is also a bit of white space in between paragraphs and other text. The code for the paragraph tags are <p> and </p>. It is important to note that the formatting of the text you type in between the paragraph tags won't be taken into account on the website. So if you type in a return inside a paragraph element, it won't show up on the website. Here is some exapmle code:

<!DOCTYPE html>
<html>
<body>
  <h1>This is a Heading</h1>
  <p>This is a paragraph</p>
  <p>This is another paragraph</p>
</body>
</html>

This exapmle code would display this on a webpage:

This is a Heading

This is a paragraph

This is another paragraph

Break

The break element is used like a return key on a normal document. It indicates a line break in between text. The code for a break element is just a single <br> with no end tag. Here is an example:

<!DOCTYPE html>
<html>
<body>
  <h1>This is a Heading</h1>
  <p>This is an example of a single paragraph<br>with a break in it</p>
</body>
</html>

This code would result in the following display:

with a break in it.

Lines

The line element is a version of a break element, and it works the same way however instead of an empty space in between lines it inserts a horizontal line for more empasis of separation. The code for this element is also jus a single <hr> tag with no end tag. Here is some example code

Comments

Comments are used as a way to take out parts of your code, so it doesn't display on the website, or to add comments to yourself so you can figure out what is going on if you come back to your code after a while. The code for a comment element is <!-- for a starting tag, and --> for the end tag with whatever you want to say in the comment in between

<!DOCTYPE html>
<html>
<body>
  <!--this code wont show up on the website-->
  <h1>This is a Heading</h1>
  <p>This is a paragraph>
</body>
</html>

Links

Adding links is very important in html becasue you can create subpages and link the homepage to each of your subpages in your website. The code for a link element is <a href ="URL of website"> and an end tag of </a>. You can type in text or even add apicture in between these tags so that when you click on the text it takes you to the specified website. A link defaults to showing up as blue underlined text on a website and a visited link shows up as purple undelined text, however you can change this using CSS styles. Here is some example code:

<!DOCTYPE html>
<html>
<body>
  <a href="https://www.w3schools.com/">Click to visit w3schools.com</a>
</body>
</html>

This code would relust in this website:

Click to visit w3schools.com

Lists

There are multiple types of lists that you can create on a webpage. The two types of lists include unordered lists and ordered lists. The code to make an unordered list is <ul> and </ul> and an ordered list is <ol> and </ol>. The lists elements are defined by the <li> and </li> tags. The list elements go inside the unordered or ordered list tags. Here is some example code:

<!DOCTYPE html>
<html>
<body>
  <ul>
    <li>Baseball</li>
    <li>Soccer</li>
    <li>Football</li>
  </ul>
  <ol>
    <li>Add sugar</li>
    <li>Mix the batter</li>
    <li>Put it in the oven</li>
  </ol>
</body>
</html>

This code would produce the following webpage

  • Baseball
  • Soccer
  • Football
  1. Add sugar
  2. Mix the batter
  3. Put it in the oven

Tables

You can also create tables using html.To define a table element, use the <table> and </table> tags. Inside each table is a set of rows. A row is defined by the <tr> and </tr> tags. Inside each row you can have two types of data. A table header, which defaults to bold and centered, and normal table data. A table header element is defined using the <th> and </th> tags while table data uses the <td> and </td> tags. You can also add styles to the table using CSS. Below is some example code:

<!DOCTYPE html>
<html>
<body>
  <table>
    <tr>
      <th>Firstname</th>
      <th>Lastname</th>
      <th>Age</th>
    </tr>
    <tr>
      <td>Jill</td>
      <td>Smith</td>
      <td>50</td>
    </tr>
    <tr>
      <td>Eve</td>
      <td>Jackson</td>
      <td>94</td>
    </tr>
    <tr>
      <td>Johon</td>
      <td>Doe</td>
      <td>80</td>
    </tr>
  </table>
</body>
</html>

This code would display the following website.

Firstname Lastname Age
Jill Smith 50
Eve Jackson 94
John Doe 80

Blocks

A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can). However, an inline element does not start on a new line and only takes up as much width as necessary.The <div> and </div> tags define a block element while the <span> and </span> tags define an inline element. The main reason to creat inline and block elements is so you can assing different style to them using CSS. Below is some example code:

<!DOCTYPE html>
<html>
<body>
  <div>This is an example</div>
  <div>of a block level element</div>
  <br>
  <span>This is an example</span>
  <span>of an inline element</span>
</body>
</html>

This code would produce this webpage:

This is an example
of a block-level element

This is an example of an inline element
